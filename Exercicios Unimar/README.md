## 📚 Lista de Exercícios - UNIMAR

Este repositório também reúne os exercícios desenvolvidos durante o curso de **Análise e Desenvolvimento de Sistemas (ADS) da UNIMAR**, com foco no aprendizado de lógica de programação e fundamentos da linguagem Python.

### 🎯 Objetivos
- Desenvolver o raciocínio lógico;
- Praticar estruturas de decisão e repetição;
- Trabalhar com entrada e saída de dados;
- Resolver problemas matemáticos utilizando Python;
- Aplicar conceitos de programação orientada a objetos (POO).

### 📝 Conteúdos Praticados
- ✅ Variáveis e Tipos de Dados
- ✅ Operadores Aritméticos e Lógicos
- ✅ Estruturas Condicionais (`if`, `elif`, `else`)
- ✅ Estruturas de Repetição (`for`, `while`)
- ✅ Funções
- ✅ Listas e Tuplas
- ✅ Manipulação de Strings
- ✅ Cálculos Matemáticos (IMC, Média, Diferença, etc.)
- ✅ Exercícios de Lógica de Programação
- 🚧 Programação Orientada a Objetos (POO)

### 📂 Exercícios Desenvolvidos
- 🔹 Cálculo de IMC
- 🔹 Média de Notas
- 🔹 Diferença entre Números
- 🔹 Operações Matemáticas
- 🔹 Verificação de Número Par ou Ímpar
- 🔹 Maior e Menor Número
- 🔹 Exercícios de Entrada e Saída de Dados
- 🔹 Problemas de Lógica propostos pela UNIMAR

### 🌱 Aprendizados
Cada exercício representa um passo na minha jornada como estudante de ADS, ajudando a construir uma base sólida em programação e resolução de problemas.

> "A prática constante transforma lógica em conhecimento e conhecimento em projetos." 💜
